## Installation

KiwiRACMatchers is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
#pod "KiwiRACMatchers"
pod 'KiwiRACMatchers', :git => 'https://github.com/PaulTaykalo/KiwiRACMatchers'
```
